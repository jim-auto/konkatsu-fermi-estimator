# データソース候補

このツールはフェルミ推定なので、すべてを厳密な統計値に置き換えることはできません。ただし、条件ごとに「公式統計に寄せられるもの」と「民間調査・仮定に留めるもの」は分けられます。

## 公式統計に寄せやすい条件

| 条件 | 使えるデータ | 主な出典 |
| --- | --- | --- |
| 年齢・性別人口 | 年齢各歳・男女別人口 | 総務省統計局「人口推計」 |
| 未婚率 | 年齢階級・男女・配偶関係 | 総務省統計局「国勢調査」、国立社会保障・人口問題研究所「出生動向基本調査」 |
| 年収 | 給与階級別・男女別分布 | 国税庁「民間給与実態統計調査」 |
| 学歴 | 最終卒業学校、進学率 | 総務省統計局「国勢調査」、文部科学省「学校基本調査」 |
| 身長・BMI | 年齢階級・男女別の身長、体重、BMI | 厚生労働省「国民健康・栄養調査」 |
| 正社員・職業 | 就業状態、正規・非正規、職業分類 | 総務省統計局「労働力調査」「就業構造基本調査」 |
| 子供希望・結婚観 | 独身者調査、夫婦調査 | 国立社会保障・人口問題研究所「出生動向基本調査」 |

## 公式統計だけでは弱い条件

| 条件 | 扱い |
| --- | --- |
| 外見レベル | 公的統計では扱いづらい。上位何%という主観条件として仮定にする。 |
| カップ数 | 公的統計は見つけにくい。ワコールなどの民間調査は参考になるが、全国代表性は限定的。 |
| 経験人数 | 公的統計で「人数分布」は見つけにくい。出生動向基本調査には性経験や交際経験に関する項目はあるが、婚活条件として使うなら強い注意書きが必要。 |
| 共働き可否 | 出生動向基本調査や労働力調査から近い指標は作れるが、「本人の希望」としては推定が必要。 |

## 実装方針

1. 年齢・性別人口、未婚率、年収、身長、学歴、就業状態は公的統計ベースに寄せる。
2. 子供希望・結婚観は出生動向基本調査を参照する。
3. 外見、カップ数、経験人数は「エンタメ寄りの仮定」として UI 上でも明示する。
4. 条件同士の相関、たとえば年収と学歴、都市部と年収、年齢と未婚率は将来的に補正する。
