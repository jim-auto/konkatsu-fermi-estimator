# データソース候補

このツールはフェルミ推定なので、すべてを厳密な統計値に置き換えることはできません。ただし、条件ごとに「公式統計に寄せられるもの」と「民間調査・仮定に留めるもの」は分けられます。

## 公式統計に寄せやすい条件

| 条件 | 使えるデータ | 主な出典 |
| --- | --- | --- |
| 年齢・性別人口 | 年齢各歳・男女別人口 | 総務省統計局「人口推計」 |
| 未婚率 | 年齢階級・男女・配偶関係 | 総務省統計局「国勢調査」、国立社会保障・人口問題研究所「出生動向基本調査」 |
| 年収 | 給与階級別・男女別分布 | 国税庁「民間給与実態統計調査」 |
| 身長・スペ値 | 年齢階級・男女別の身長、体重 | 厚生労働省「国民健康・栄養調査」 |

## 公式統計だけでは弱い条件

| 条件 | 扱い |
| --- | --- |
| 外見偏差値 | 公的統計では扱いづらい。主観評価を偏差値風に置き換えた仮定にする。 |
| カップ数 | 公的統計は見つけにくい。ワコールなどの民間調査は参考になるが、全国代表性は限定的。 |
| 経験人数 | 公的統計で「人数分布」は見つけにくい。出生動向基本調査には性経験や交際経験に関する項目はあるが、婚活条件として使うなら強い注意書きが必要。 |

## 実装方針

1. 年齢・性別人口、未婚率、年収、身長は公的統計ベースに寄せる。
2. 外見偏差値、カップ数、経験人数は「エンタメ寄りの仮定」として UI 上でも明示する。
3. 条件同士の相関、たとえば都市部と年収、年齢と未婚率は将来的に補正する。

係数ごとの確認状況は `docs/fact-check.md` に分けて記録します。

## 現在コードに反映したもの

| 条件 | 反映内容 |
| --- | --- |
| 基本人口 | 総務省統計局「人口推計 2024年10月1日現在」の総人口・男女別人口 |
| 年齢レンジ人口 | 人口推計2024年の各歳人口を10代後半、20代前半、20代後半のような年齢帯へ合算 |
| 未婚率 | 2020年国勢調査の5歳階級別未婚率を対象レンジに合わせて加重平均 |
| 年収 | 国税庁「令和6年分 民間給与実態統計調査」第3表の男性給与階級から累積比率を算出 |
| 身長 | 国民健康・栄養調査の男性平均身長・標準偏差をもとに正規近似 |
| スペ値 | 国民健康・栄養調査の若年女性の身長・体重を参考に、身長(cm)-体重(kg)の近似値へ調整 |
