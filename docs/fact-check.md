# ファクトチェック状況

最終確認日: 2026-04-16

このドキュメントは、アプリ内の係数がどの程度ファクトチェック済みかを切り分けるためのメモです。  
このツールはフェルミ推定なので、すべての値を厳密な統計値として扱うのではなく、以下の4段階で管理します。

| 区分 | 意味 |
| --- | --- |
| 確認済み | 公的統計の公開値とおおむね一致している |
| 近似 | 公的統計を参考にしているが、集計・補間・正規近似などを含む |
| 仮定 | 公的統計で直接扱いづらく、UX上の仮定として置いている |
| 要再検証 | 根拠が弱い、または今後の再計算が必要 |

## 係数別ステータス

| 条件 | 現在の扱い | ステータス | 根拠・注意 |
| --- | --- | --- | --- |
| 日本人口・男女別人口 | `123,802,000`、男性 `60,233,000`、女性 `63,569,000` | 確認済み | 総務省統計局「人口推計 2024年10月1日現在」を参照。 |
| 年齢帯人口 | 10代後半、20代前半など | 近似 | 人口推計をベースにする方針は妥当。ただし現在値は一部を年数按分・丸めで置いているため、各歳データからの再集計ではない。 |
| 未婚率 | 年齢帯・男女別に設定 | 近似 | 2020年国勢調査の配偶関係データを参照する方針は妥当。ただし現在値は5歳階級からの厳密再集計ではなく、近似値。 |
| 居住地 | 都市部 `56%`、地方 `44%` | 仮定 | 都市部の定義が未固定。三大都市圏、政令市、DID人口など、どの定義を使うかで変わる。 |
| 年収 | 男性年収400万以上〜1200万以上 | 近似 | 国税庁「令和6年分 民間給与実態統計調査」を参照。給与所得者ベースであり、全男性・年齢別・婚活市場参加者とは一致しない。 |
| 身長 | 男性身長165cm以上〜180cm以上 | 近似 | 国民健康・栄養調査の身長分布を参考にした正規近似。年齢別差や測定年の差は簡略化。 |
| スペ値 | `身長(cm) - 体重(kg)` 90以上〜120以上 | 仮定 | 身長・体重の公的統計はあるが、スペ値そのものの分布は公的統計ではない。相関を単純化したエンタメ寄りの条件。 |
| 外見偏差値 | 50以上〜70以上 | 仮定 | 公的統計では扱えない主観条件。正規分布風の見せ方をしているが、実測分布ではない。 |
| カップ数 | A-B〜H以上 | 仮定 | ワコール等の民間調査は参考になるが、全国代表性や年齢別分布は限定的。現在値は自己申告前提の仮定。 |
| 経験人数 | 0人〜21人以上 | 要再検証 | 自己申告バイアスが大きく、婚活条件として使うには特に注意が必要。現在値はエンタメ寄りの仮定。 |
| 平均条件との比較 | アプリ内プリセット比較 | 仮定 | 「平均条件」は統計上の平均ではなく、比較用のゆるめプリセット。 |

## 現時点の結論

このアプリの数値は、以下の混合です。

- 公的統計に近い値: 基本人口
- 公的統計を参考にした近似: 年齢帯人口、未婚率、年収、身長
- エンタメ寄りの仮定: 居住地、外見偏差値、スペ値、カップ数、経験人数

そのため、UIやREADMEでは「推定」「近似」「仮定」を明示し、実際の人口や個人の可能性を断定しない表現にします。

## 今後の再検証タスク

1. 人口推計の各歳・男女別CSV/Excelから、年齢帯人口を再集計する。
2. 2020年国勢調査の配偶関係データから、年齢帯ごとの未婚率を再集計する。
3. 国税庁の給与階級表から、年収条件の累積比率計算表を `docs/` に残す。
4. 身長条件は年齢階級別に補正する。
5. 都市部の定義を固定し、国勢調査または人口集中地区データで再計算する。
6. 外見偏差値、カップ数、経験人数は「統計値」ではなく「仮定」としてUIにも表示する。
