# 条件パターン検証

## 極端: 男性対象

条件:

- アラサー（25-34歳）
- 都市部
- 未婚
- 年収800万以上
- 身長175cm以上
- 大卒以上
- 安定職

計算イメージ:

```text
6,644,000
x 0.638
x 0.56
x 0.184
x 0.27
x 0.46
x 0.38
= 約20,600人
```

年収800万円以上は、国税庁の男性給与階級では約18.4%です。以前の仮定より少し広くなりましたが、身長・学歴・職業を重ねると母集団は一気に減ります。

## 極端: 女性対象

条件:

- アラサー（25-34歳）
- 都市部
- 未婚
- 外見上位10%
- スペ値105以上
- Dカップ
- 大卒以上
- 正社員
- 共働き希望
- 子供を希望
- 経験人数1-2人

計算イメージ:

```text
6,270,000
x 0.518
x 0.56
x 0.10
x 0.32
x 0.18
x 0.42
x 0.42
x 0.68
x 0.58
x 0.22
= 約160人
```

外見、スペ値、カップ数、経験人数のような主観・自己申告条件を積むと、フェルミ推定上の母集団はほぼ消えます。ここは厳密な統計ではなく、条件を足す怖さを見せるためのエンタメ寄りの部分です。

## 今後の検証候補

- 年収と学歴の相関補正
- 都市部と年収の相関補正
- 年齢ごとの子供希望率
- 女性対象のスペ値・カップ数・経験人数の仮定値調整
- 初婚だけでなく離別・再婚意向を含めるモード
- 都道府県別の母集団
