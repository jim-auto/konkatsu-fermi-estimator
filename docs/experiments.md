# 条件パターン検証

## 極端: 男性対象

条件:

- 30-39歳
- 都市部
- 未婚
- 年収800万以上
- 身長175cm以上
- 大卒以上
- 安定職

計算イメージ:

```text
60,500,000
x 7,000,000 / 60,500,000
x 0.41
x 0.56
x 0.14
x 0.26
x 0.48
x 0.38
= 約5,900人
```

各条件は単体では極端に見えなくても、同時に満たす人数は大きく減ります。

## 極端: 女性対象

条件:

- 25-34歳
- 都市部
- 未婚
- 外見上位10%
- 細身
- 正社員
- 共働き希望
- 子供を希望

計算イメージ:

```text
63,500,000
x 6,400,000 / 63,500,000
x 0.47
x 0.56
x 0.10
x 0.33
x 0.42
x 0.68
x 0.58
= 約9,600人
```

外見や体型のような主観条件を入れると、母集団は急速に少なくなります。

## 平均条件: 男性対象

条件:

- 30-39歳
- 都市部
- 未婚
- 年収400万以上
- 身長170cm以上
- 大卒以上
- 正社員

狭めすぎない条件でも、全人口から見ると数%未満の母集団になります。アプリ内では現在条件とこの平均条件を比較し、絞り込みの強さを直感的に確認できます。

## 今後の検証候補

- 年収と学歴の相関補正
- 都市部と年収の相関補正
- 年齢ごとの子供希望率
- 初婚だけでなく離別・再婚意向を含めるモード
- 都道府県別の母集団
