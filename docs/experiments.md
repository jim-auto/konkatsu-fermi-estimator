# 条件パターン検証

## 極端: 男性対象

条件:

- 10代後半〜20代前半（18-24歳）
- 都市部
- 未婚
- 顔面偏差値65以上
- スペ値105以上
- 大卒以上
- 年収800万以上
- 身長175cm以上

計算イメージ:

```text
4,365,000
x 0.967
x 0.56
x 0.07
x 0.32
x 0.52
x 0.184
x 0.27
= 約1,400人
```

外見・体型・学歴・年収・身長を同時に重ねると、10代後半〜20代前半・都市部・未婚の母集団からさらに大きく減ります。
