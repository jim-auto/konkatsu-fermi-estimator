このツールはフェルミ推定です。公的統計に寄せられる係数は実データを参照し、公式統計が弱い条件は「仮定」として明示します。

## 計算式

```text
推定人数 = 対象性別人口
  x 年齢レンジ割合
  x 未婚率
  x 居住地割合
  x 条件1割合
  x 条件2割合
  ...
```

条件同士の相関は厳密には大きい可能性があります。たとえば都市部と年収、年齢と未婚率は独立ではありません。このツールではUXと納得感を優先し、相関補正はまだ入れていません。

## 注意が必要な条件

外見偏差値、スペ値、学歴、カップ数、経験人数は婚活UXとしては入力したくなる条件ですが、公式統計として扱いにくい項目です。特に外見偏差値、難関大卒以上のような学校群、カップ数、経験人数は主観評価や自己申告バイアスが大きいため、このツールではエンタメ寄りの仮定として扱います。
