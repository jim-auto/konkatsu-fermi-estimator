## 計算式

```text
推定人数 = 対象性別人口
  x 年齢レンジ割合
  x 未婚率
  x 居住地割合
  x 条件1割合
  x 条件2割合
  ...
```

条件同士の相関は厳密には大きい可能性があります。たとえば年収と学歴、都市部と年収、職業と年収は独立ではありません。このツールではUXと納得感を優先し、相関補正はまだ入れていません。

## 注意が必要な条件

外見、スペ値、カップ数、経験人数は婚活UXとしては入力したくなる条件ですが、公式統計として扱いにくい項目です。特にカップ数と経験人数は自己申告バイアスが大きいため、このツールではエンタメ寄りの仮定として扱います。
